## Project Overview

This project implements a **Data Warehouse system** to analyze hotel booking data using OLAP techniques. The goal is to understand customer behavior, booking patterns, cancellation trends, and revenue contribution to support data-driven decision making in the hospitality industry.

## Methods

The implemented process consists of:

* Data Collection
* Data Cleaning & Transformation (ETL)
* Data Warehouse Modeling
* OLAP Analysis
* Data Visualization

## Dataset

Hotel Booking Dataset from Kaggle:
[(https://www.kaggle.com/datasets/namit18/hotel-booking-demand)]

## Analysis Highlights

* City Hotel has a higher number of bookings compared to Resort Hotel
* Online Travel Agent (OTA) dominates booking sources
* City Hotel has a higher cancellation rate
* OTA contributes the highest revenue
* Customer behavior varies based on country and booking channel
